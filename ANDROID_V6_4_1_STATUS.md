# ConnectChat Pro Android v6.4.1

The earlier Android work contained only a Capacitor configuration. This release
now includes a complete generated Android Studio project under:

`clients/android/android`

## Included

- Application ID: `com.connectchat.pro`
- Application name: `ConnectChat Pro`
- Version name: `6.4.1`
- Version code: `641`
- Secure remote application URL:
  `https://connectchat-pro-kanaan.onrender.com`
- HTTPS-only network policy
- Internet permission
- Camera permission for photos and video calls
- Microphone and audio permissions for voice notes and calls
- File chooser support through the Android system picker
- Debug APK and release AAB build commands

## Not included

- A private release signing keystore
- Google Play account credentials
- Firebase configuration for native push notifications

Those items belong to the application owner and must not be included in a
shared source archive.
