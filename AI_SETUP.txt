CONNECTCHAT HYBRID AI SETUP — v5.2
==================================

HYBRID MODE — OLLAMA FIRST, OPENAI FALLBACK
--------------------------------------------
Add these server environment variables:

   AI_ENABLED=true
   AI_PROVIDER=hybrid
   AI_DEFAULT_PROVIDER=ollama
   OPENAI_API_KEY=your_private_OpenAI_API_key
   OPENAI_MODEL=gpt-4.1-mini
   OLLAMA_URL=https://your-secured-ollama-server.example.com
   OLLAMA_MODEL=qwen2.5:7b
   AI_REQUEST_TIMEOUT_MS=120000

In ConnectChat AI, users can select Auto, Ollama, or OpenAI. Auto tries the
default provider first and uses the other configured provider if it fails.

IMPORTANT FOR RENDER
--------------------
OLLAMA_URL=http://127.0.0.1:11434 does not connect Render to Ollama on your
Windows PC. Render requires an HTTPS address for a separately hosted, secured
Ollama server. Never publish an unprotected Ollama endpoint on the internet.

OPTION A — OPENAI (RECOMMENDED FOR RENDER / WEB / MOBILE)
---------------------------------------------------------
1. Open your hosting service environment variables.
2. Add:
   AI_ENABLED=true
   AI_PROVIDER=openai
   OPENAI_API_KEY=your_private_OpenAI_API_key
   OPENAI_MODEL=gpt-4.1-mini
3. Restart or redeploy the server.
4. Log in. "ConnectChat AI" will appear at the top of the contacts list.

OPTION B — OLLAMA (FREE LOCAL AI)
---------------------------------
1. Install Ollama on the same computer that runs server.js.
2. Run: ollama pull qwen2.5:7b
3. Add to .env:
   AI_ENABLED=true
   AI_PROVIDER=ollama
   OLLAMA_URL=http://127.0.0.1:11434
   OLLAMA_MODEL=qwen2.5:7b
4. Restart ConnectChat.

Important: Render cannot normally reach Ollama running on your private PC.
Use OpenAI for a public Render deployment, or host Ollama on a reachable,
secured server.

SECURITY
--------
- Never put OPENAI_API_KEY in public/app.js, index.html, GitHub, or screenshots.
- The key is used only by server.js.
- AI requests require a logged-in and approved ConnectChat account.
- The server applies an AI rate limit.

CURRENT AI FEATURES
-------------------
- Text questions and answers
- Arabic and English
- Short conversation memory saved in the user's browser
- MEP/HVAC/construction and general assistance
- Real conversation summaries
- Action-item extraction
- Latest-message Arabic/English translation
- Visible provider/model status and useful setup errors
- OpenAI and Ollama provider support
- Auto / Ollama / OpenAI selector
- Ollama-first automatic fallback to OpenAI
- Provider and model label on each response

NOT INCLUDED IN THIS RELEASE
----------------------------
- AI analysis of uploaded files, images, or voice notes
- Cloud-synced AI conversation history

These can be added later after the basic AI text service is tested and the API billing limit is configured.
