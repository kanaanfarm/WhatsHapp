CONNECTCHAT PRO ENTERPRISE v5

1. Upload all project files to GitHub/Render.
2. Open Supabase > SQL Editor.
3. Run enterprise-v5-migration.sql once.
4. Confirm your existing .env values are still configured.
5. Optional AI: add OPENAI_API_KEY and set AI_ENABLED=true.
6. Deploy latest commit.
7. Refresh the browser with Ctrl+Shift+R.

IMPLEMENTED IN THIS RELEASE
- Server-backed Groups with persistent messages.
- Server-backed Channels with persistent posts and announcements.
- Server-backed Files index for private-chat attachments.
- Server-backed Call History.
- Existing private chat, profiles, status, uploads, AI endpoint and WebRTC calls retained.
- Theme and accent remain inside Settings.

LIMITATIONS
- Group member invitation UI is not yet included; the API/database foundation supports members.
- Group video meetings and call recordings are not included.
- AI requires a valid server API key.
- File folders, version history and permissions are future modules.
