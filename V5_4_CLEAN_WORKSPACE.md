# ConnectChat Pro Enterprise v5.4

## Cleaner conversation sidebar

- Manage users, Status, and Recovery code no longer appear beneath contacts.
- These functions remain available from Settings.
- The conversation list receives the recovered vertical space.

## Corrected chat controls

- Smart Actions now occupy a dedicated toolbar above the composer.
- The toolbar is hidden until a conversation is selected.
- The toolbar no longer overlaps the message composer.

## User-controlled composer

Every account can choose **Settings → My page appearance → Message-bar icons**:

- **Hide extra icons** keeps only the message field and Send button.
- **Show all icons** restores emoji, attachment, camera, and microphone.

The clean essential composer is the default. No database migration is required.
