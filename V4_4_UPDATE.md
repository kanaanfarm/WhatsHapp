# ConnectChat Pro v4.4 update

Implemented in this package:

- Fixed, always-visible account panel at the bottom of the conversation sidebar.
- Always-visible Logout button.
- Switch Account action, which securely logs out and returns to the login screen.
- Account name, initials and administrator/member role shown in the fixed panel.
- Professional message composer with emoji picker, file attachment, camera upload, voice recording and send button.
- Multiline message input: Enter sends; Shift+Enter creates a new line.
- Auto-expanding message field.
- Mobile layout adjustments.
- Updated browser/service-worker cache version so deployed clients receive the changes.
