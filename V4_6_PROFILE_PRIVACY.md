CONNECTCHAT PRO v4.6 - PROFILE PRIVACY

- Every approved user can be viewed from the chat list or active chat header.
- Clicking a user's avatar opens a separate profile page.
- Other users' profiles are view-only.
- Only the signed-in account owner can upload, replace, or remove their own profile photo.
- The server avatar endpoints always update the authenticated user's own account only.
- Profile pages provide Message, Voice Call, and Video Call actions for other users.
- The chat page remains focused on messaging.
