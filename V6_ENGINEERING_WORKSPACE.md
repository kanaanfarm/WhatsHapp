# ConnectChat Pro Enterprise v6

## Responsive workspace

The interface now adapts when the browser is resized, minimized, maximized, or
zoomed. Panels reduce progressively, the overview hides when space is limited,
and messages retain usable width. Browser zoom remains available for
accessibility.

## Profile photos

Avatar links now remain valid for seven days per application session. Images
use consistent centering and inherited corner shapes. Each account can choose
**Crop to fill** or **Fit full photo** in My page appearance.

## AI exports

Each successful AI answer includes:

- Word export (`.docx`)
- PDF export (`.pdf`)
- Excel export (`.xlsx`)

Exports are generated by the authenticated ConnectChat server. Exporting an
existing answer does not make another AI request.

## Shared calculation sheets

The new **Sheets** workspace allows every approved user to:

- Upload XLSX, XLS, CSV, or PDF calculation sheets
- Add a title and description
- See uploader, date, and file size
- Download files through an authenticated endpoint
- Delete their own uploads

Administrators may delete any shared sheet.

Run `v6-calculation-sheets-migration.sql` once before using this workspace.
