ConnectChat Pro v3.2

Included:
- Audio and video calls through WebRTC
- In-call screen sharing from the Share screen button
- Share an entire display, application window, or browser tab (browser options vary)
- Stop sharing and return to the camera without ending the call
- Remote screen-sharing status
- Last-message preview, time, unread count, typing preview, and newest-chat sorting

How to use screen sharing:
1. Log in as two approved users in two different browsers or devices.
2. Select the other user.
3. Start a VIDEO call using the camera button.
4. The recipient accepts the call.
5. During the active video call, click "Share screen".
6. Choose Entire Screen, Window, or Browser Tab in the browser prompt.
7. Click "Stop sharing" to return to the camera.

Important:
- Screen sharing is visible only inside an active VIDEO call.
- Production deployment must use HTTPS. Render provides HTTPS automatically.
- Keep Render environment variables configured; this package intentionally excludes .env.
- A TURN server is strongly recommended for reliable calls across different networks.
