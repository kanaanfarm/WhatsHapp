# ConnectChat Pro v4 Workspace

Implemented from the approved proposal:

- Full dark left navigation rail
- Modern workspace conversation list
- Filters and quick-contact row
- Updated chat header, canvas, composer and right insights panel
- Existing authentication, admin approval, Supabase storage and private messaging retained
- Audio/video calls retained
- Real WebRTC screen sharing added inside video calls
- Share entire screen, application window or browser tab
- Stop sharing and return to camera without ending the call
- Remote screen-sharing status

## Screen sharing

Select a user, start a video call, wait for acceptance, then press **Share screen** in the call window. Screen sharing requires HTTPS when deployed.
